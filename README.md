# Procesamiento de datos a gran escala
En este repositorio de andan compilados una serie de documentos que tienen como objetivo el procesamiento de datos a gran escala.

## Autor
- Diego Alejandro Jara Rojas

## Contenido del repositorio

- **Análisis-Premier-league** → Análisis estadístico de datos de la Premier League.  
- **Classes-Basicas-Python-main** → Ejercicios básicos para repasar programación en Python.  
- **AnálisisCalidadAguaIndia.ipynb** → Notebook para el análisis de calidad del agua en India usando modelos de Machine Learning.  
- **LABORATORIO SPARK ACOSTA GONZALEZ GU...** → Práctica de procesamiento de datos distribuidos con Apache Spark.

## Tecnologías utilizadas

- **Python 3.10+**
- **Pandas**, **NumPy**, **Matplotlib**, **Seaborn**
- **TensorFlow / Keras** para modelos de aprendizaje automático
- **Apache Spark** para procesamiento distribuido
- **Jupyter Notebook** para análisis interactivo
